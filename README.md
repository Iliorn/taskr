# taskr

A terminal task manager built with Go and Bubbletea.

## Installation

```
git clone https://github.com/luciphere/taskr
cd taskr
go mod tidy
go build -o taskr .
```

Move the binary somewhere on your PATH:

```
mv taskr ~/.local/bin/
```

## Self-update

Press `U` inside the app to update to the latest release.
